# [`CALM`](https://github.com/cwendorf/CALM)

## Reference

This section is the reference for `CALM`. Each page documents one exported function, including usage, arguments, and return values.

### Coding and Decoding Matrices

- [calm.decode](./calm.decode.md): Recovers the hypothesis matrix from a coded contrast matrix.
- [calm.encode](./calm.encode.md): Transforms a hypothesis matrix L into its corresponding coding matrix.

### Specific Contrast Codes

- [control.first](./control.first.md): Contrast coding with the first level as the control group.
- [control.last](./control.last.md): Contrast coding with the last level as the control group.
- [deviation.first](./deviation.first.md): Deviation contrast coding with the first level as the reference.
- [deviation.last](./deviation.last.md): Deviation contrast coding with the last level as the reference.
- [difference.first](./difference.forward.md): Forward difference contrast coding.
- [difference.reverse](./difference.reverse.md): Reverse difference contrast coding.
- [helmert.foward](./helmert.forward.md): Forward Helmert contrast coding.
- [helmert.reverse](./helmert.reverse.md): Reverse Helmert contrast coding. 
- [polynomial](./polynomial.md):Orthogonal polynomial contrast coding. 
- [repeated.forward](./repeated.forward.md): Repeated forward contrast coding.
- [repeated.reverse](./repeated.reverse.md): Repeated reverse contrast coding.
- [treatment.first](./treatment.first.md): Treatment contrast coding using the first level as the baseline.
- [treatment.last](./treatment.last.md): Treatment contrast coding using the last level as the baseline.
